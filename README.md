# CODESOFT - Data Science Internship

This repository contains the Data Science projects completed as part of the CodSoft Internship.

## Projects

### 1. Titanic Survival Prediction
- Predicts passenger survival using Machine Learning.

### 2. Movie Rating Prediction
- Predicts movie ratings using regression techniques.

### 3. Iris Flower Classification
- Classifies Iris flowers into different species.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
  
 ## Machine Learning Workflow
- Data Cleaning
- Missing Value Handling
- Feature Engineering
- One-Hot Encoding
- Train-Test Split
- Random Forest Classifier
- Model Evaluation
