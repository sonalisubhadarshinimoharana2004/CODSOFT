# CODESOFT - Machine Learning Internship

This repository contains the Machine Learning projects completed as part of the CodSoft Internship.

## Projects

### 1. Titanic Survival Prediction
- Predicts passenger survival using Machine Learning.

### 2. Movie Rating Prediction
- Predicts movie ratings using regression techniques.

### 3. Iris Flower Classification
- Classifies Iris flowers into different species.

- NumPy
- Scikit-learn
- Matplotlib
